**A Gentle Introduction to Optimization** exercises
====

This repo holds exercises for
[*A Gentle Introduction to Optimization*](https://www.amazon.com/Gentle-Introduction-Optimization-B-Guenin/dp/1107658799).

Exercise files that require typesetting only will be in `typst` (for practice/for getting used to
match typesetting in `typst`).
Exercise files where I get to write any code will all be in Haskell (for _joy_).
